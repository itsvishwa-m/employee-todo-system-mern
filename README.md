# Employee Todo Management System (MERN Stack)

## Overview
A full-stack Employee Todo Management System built using the MERN Stack.

## Features

- User Login Authentication
- Create Todo
- View Todos
- Mark Todo as Completed
- Delete Todo
- JWT Authentication
- MongoDB Atlas Integration
- Responsive UI
- Render Deployment

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JWT
